# Jingle-Bells-Microbit



1. Connect battery to microbit.
2. Attach buzzer in pin 0.
3. Attach vibrator in pin 2.
4. Attach rotary angle sensor to pin 1.


Materials:
Vibrator
Buzzer
Rotary Angle Sensor
Microbit
Battery
